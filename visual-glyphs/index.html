SBOL Visual Glyphs

SBOL Visual is built off of the latest specification release. These glyphs are made to be drawable on a white board but they are not definite, they are guidelines. Sample visual glyphs are supported in PDF, PNG, SVG. Visit our GitHub repository to request features and report issues related to SBOL Visual. Take a look at the latest software tools that support SBOL Visual glyphs. Contact the SBOL Visual team for further questions.

Look at this quick introduction to SBOL Visual 2.1 to get started.

Zip files containing all glyphs as PDF, PNG, or SVG can be downloaded from the Releases page of the SBOL Visual GitHub repository.
