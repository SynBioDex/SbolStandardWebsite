## Build with SBOL

SBOL provides well-defined data models, formal specifications, and libraries in multiple programming languages. Whether you're building a design tool, a repository, or an analysis pipeline, SBOL gives you a standard way to represent and exchange genetic designs.

---

## Programming Libraries

The SBOL community maintains actively developed libraries for five languages. All support SBOL 3.x.

### Python — pySBOL3

The most widely used SBOL library. Clean, Pythonic API for creating, reading, and manipulating SBOL documents.

{{< cta cta_text="pySBOL3 on GitHub" cta_link="https://github.com/SynBioDex/pySBOL3" cta_new_tab="true" >}}
{{< cta cta_text="Documentation" cta_link="https://pysbol3.readthedocs.io" cta_new_tab="true" >}}

### Java — libSBOLj

Full-featured Java library for SBOL. Used in SBOLDesigner, SBOLCanvas, and other production tools.

{{< cta cta_text="libSBOLj on GitHub" cta_link="https://github.com/SynBioDex/libSBOLj" cta_new_tab="true" >}}

### C++ — libSBOL

C++ library for reading and writing SBOL documents, suited for performance-critical applications and tool integration.

{{< cta cta_text="libSBOL on GitHub" cta_link="https://github.com/SynBioDex/libSBOL" cta_new_tab="true" >}}

### JavaScript — sbol.js

JavaScript library for working with SBOL in Node.js and browser environments.

{{< cta cta_text="sbol.js on GitHub" cta_link="https://github.com/SynBioDex/sbol-javascript" cta_new_tab="true" >}}

### F# — libSBOLcs

.NET/F# library providing SBOL support for the Microsoft ecosystem.

{{< cta cta_text="libSBOLcs on GitHub" cta_link="https://github.com/SynBioDex/libSBOLcs" cta_new_tab="true" >}}
