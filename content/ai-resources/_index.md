## Tips for Using AI with SBOL

- **Be specific.** Mention which version of SBOL you are working with (SBOL2 or SBOL3) and which library (e.g. pySBOL3, libSBOLj).
- **Paste your code or SBOL document.** The AI can help diagnose issues if it can see the actual data.
- **Cross-reference.** Confirm any specification details against the [official PDF](/docs/SBOL3.0.1.pdf) or the [GitHub repository](https://github.com/SynBioDex).
- **Ask for examples.** These tools are good at generating short, working code snippets.
